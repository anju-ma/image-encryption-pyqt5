# Image Encryption System (Python)

## Description
A Python-based image encryption and decryption application built with PyQt5.
The system includes user authentication with SHA-256 password hashing and
stores user credentials securely using SQLite.

## Features
- User login and registration
- SHA-256 password hashing
- Image encryption and decryption
- PyQt5 graphical user interface
- SQLite database for user management

## Technologies Used
- Python
- PyQt5
- SQLite
- Cryptography
